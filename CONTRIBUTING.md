# Contributing

Thank you for your interest in contributing to this project!

## How to Contribute

1. **Report Bugs** — Open an issue describing the bug, steps to reproduce, and expected behavior.
2. **Suggest Features** — Open an issue with your idea and use case.
3. **Submit Pull Requests** — Fork the repo, create a branch, make your changes, and submit a PR.

## Pull Request Guidelines

- Keep PRs focused on a single change
- Include a clear description of what and why
- Add tests if applicable
- Ensure existing tests pass

## Code of Conduct

Be respectful and constructive in all interactions.
